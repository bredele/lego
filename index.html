Powerful. Fast. Easy to use.

Maple makes developping large scale applications as easy as a todo list.

Efficient

Maple.js is not only small but also really fast. It has been conceived around this simple question: why get complicated?

Less code

Write dramatically less code with Maple's live binding and plugin engine. Like jQuery you can reuse or create your own plugins.

Maintainable

Maple.js has been built with enterprise needs in mind. Out of the box, it comes in the form of extensible components with an architectural framework, test environment, guidelines and mores.

Easy to learn

No scatterbrained concept. Maple.js is as simple as old plain JavaScript and instead reinventing the wheel you can focus on your job, make an awesome app!
